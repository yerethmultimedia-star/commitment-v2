# Product Review Template

## Value Verification (Rule #95)

- [ ] **¿El usuario percibe valor inmediato?** (Describe cómo la entrega aporta valor directo y utilizable al usuario)
- [ ] **¿La funcionalidad puede demostrarse en menos de 2 minutos?** (Describe el flujo de demostración rápido y conciso)
- [ ] **¿Reduce fricción o aumenta motivación?** (Detalla los elementos visuales, microcopia o flujos que mejoran la motivación o reducen la fricción cognitiva)
- [ ] **¿Hace que el producto sea más fácil de recomendar?** (Identifica qué hace a la característica atractiva o digna de compartirse)

## Functional Verification

- [ ] Does the implementation meet the original goals?
- [ ] Are all business rules verified?

## UX / API Verification

- [ ] Are API endpoints and contracts correct?
- [ ] Is input validation working as expected?
- [ ] Are all visual strings and instructions fully localized?
